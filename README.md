# Estudo de Arquiteturas de Software com Diagramas de Sequência

Este repositório contém a análise de padrões de arquitetura (MVC, MVP, MVVM) utilizando Diagramas de Sequência como ferramenta de visualização de fluxo de mensagens.

## Objetivos
- Compreender o comportamento dinâmico dos sistemas.
- Relacionar o fluxo de mensagens com responsabilidades arquiteturais.
- Identificar padrões e anti-padrões de design.
